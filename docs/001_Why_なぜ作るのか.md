# なぜVideo Loop OSを作るのか

Video Loop OSは、自分自身のために始めた挑戦です。

私は、自分のためになることが、世界のためにもなっていると信じられる自分になりたい。

そのために、自分が本当に価値があると思えるものを作りたい。

もしその挑戦が、多くの人の役に立つのであれば、それ以上に嬉しいことはありません。

Video Loop OSは、動画投稿を自動化するための会社ではありません。

AIと人が一緒に成長し、新しい価値を生み出し続ける会社です。

私たちは、毎日実験し、毎日学び、毎日進化します。

その積み重ねが、自分のためになり、世界のためにもなると信じています。
